# Cleopatra Dashboard

> The chatter interface for Cleopatra AI.

---

## What it does

Real-time dashboard for OnlyFans agency chatters. 
Shows fan conversations, AI-generated reply suggestions, 
and fan intelligence — all in one premium dark interface.

## Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 14 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS + custom CSS |
| Realtime | Supabase Realtime |
| AI Backend | Cleopatra API on Railway |
